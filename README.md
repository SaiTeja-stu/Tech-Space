TechSpace Hub is a modern AI tools directory and admin-managed workspace built with React, TypeScript, Firebase, and Framer Motion.
It provides curated sections for Frontend, Backend, Tech Stack, Deployment, Image, Video, Audio, Editing, and PPT Generation tools, with smooth UI and real-time data flow.

Key Features
Firebase-backed tool management (Firestore + Auth)
Admin Panel with publish workflow for adding, editing, deleting, and organizing tools
Category-wise tool pages with manual + AI segmentation (including PPT tools)
Role-based admin access
Tool history tracking per user
Responsive, animated UI with premium dark theme
Static-host ready deployment (InfinityFree compatible)
